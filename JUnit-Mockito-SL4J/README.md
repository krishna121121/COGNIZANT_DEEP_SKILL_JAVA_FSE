# JUnit, Mockito & SLF4J Solutions

Maven project with solutions for all exercises in `JUnit, Mockito and SL4J/`.

## Run tests

```bash
cd solutions/JUnit-Mockito-SL4J
mvn test
```

## Package layout

| Folder | Exercise doc |
|--------|----------------|
| `junit/basic` | 1. JUnit_Basic Testing Exercises |
| `junit/advanced` | 2. JUnit_Advanced Testing exercices |
| `mockito/basic` | 3. Mockito exercises |
| `mockito/advanced` | 3. Mockito_Advanced exercises |
| `spring` | 4. JUnit_Spring Test exercises, 5. Mockito_Mock dependencies |
| `logging` | 6. SL4J Logging exercises |
