# Cognizant Digital Nurture 4.0 – Java Full Stack Engineer (FSE) Deep Skilling

## Overview

This repository contains all hands-on exercises, assignments, and learning materials completed during the **Cognizant Digital Nurture 4.0 (DN 4.0) Java Full Stack Engineer Deep Skilling Program**.

The program covers a comprehensive set of technologies and concepts required for modern Full Stack Development, including Java, Spring Boot, Microservices, Databases, Frontend Frameworks, Testing, Version Control, and Software Engineering Principles.
